# twitch-status

CLI app to check if the channels you follow on twitch are online

![twis](https://cloud.githubusercontent.com/assets/15200302/19445912/9afb4aca-9496-11e6-8b58-d9c344bc4037.png)

### manual install

```sh
git clone https://github.com/bnto/twitch-status.git
npm install -g ./twitch-status
```

You need Node 4 or greater on your machine. 

### usage

`twis [YOUR USERNAME]`
